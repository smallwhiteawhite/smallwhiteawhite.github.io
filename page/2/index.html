计算机网络概述1. 概念 计算机网络就是互联的、自治的计算机系统的集合 2. 计算机网络的组成2.1 从组成部分上计算机网络 = 硬件 + 软件 + 协议 硬件： 主机，链路 and so on。 软件： 就是各类软件。 协议：计算机网络的核心，可以近似理解成一种规则、算法。 2.2 从工作方式上计算机网络 = 边缘部分 + 核心部分 边缘部分：主要是主机之间的连接，实现通信与数据共享 核心部分：网络+连接网络的路由器，为边缘部分提供连通和交换服务 2.3 从功能组成上计算机网络 = 通讯子网 + 资源子网 通信子网：实现通信 资源子网：实现资源共享 3. 计算机网络的功能 功能 1.数据通信2.资源共享3.分布式处理：避免一个网络中很多空闲一个过忙的情形4.提高可靠性：网络中的计算机互为替代机5.负载均衡：和分布式处理有点像 4. 计算机网络的分类4.1 从分布范围上 广域网 城域网 局域网 个人区域网 4.2 从传输技术上 广播式网络所有联网的计算机共享一个公共信道，一个发报文全部都能”听到“报文。 点对点网络每条链路连接两台计算机；如果 ...
