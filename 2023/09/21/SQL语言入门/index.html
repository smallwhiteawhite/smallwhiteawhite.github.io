SQL语言入门

1. 了解SQL

1.1 概念

SQL 是用于访问和处理数据库的标准的计算机语言。
SQL 大小写不敏感。

2. SQL基本操作

2.1 选择语句

USE [DATABASE];          //选择名为database的数据库
SELECT * FROM TABLENAME; //从名为tablename表中选择所有数据         
SELECT C1,C2 FROM TABLENAME; //从名为tablename的表中选择c1,c2两列
SELECT DISTINCT C1,C2 FROM TABLENAME;//从名为tablename的表中选择c1,c2两列中不重复的元素

2.2 条件语句

WHERE 关键字:       
SELECT C1, C2, ...
FROM table_name
WHERE condition;//选择满足condition的部分

2.3 运算符

=  //等于
<>  //不等于      
>=  //大于等于         
<=  //小于等于
BETWEEN [a,b]  //在a和b之间  
AND  //与        
OR  //或

2.4 排序语句

ORDERBY 关键字:       
SELECT C1, C2, ...
FROM TABLENAME
ORDER BY C1, C2, ... ASC|DESC;
// ASC是升序排列，DESC是降序排列，默认是升序 
//这句话的意思是，先按照C1排序，得到几个子集，然后在子集里面按照C2排序

2.5 增加语句

INERT INTO 关键字                
CREATE 关键字         

给表中增加记录:    

INSERT INTO TABLENAME
VALUES(v1,v2,...)//隐式的插入       

INSERT INTO TABLENAME(c1,c2,...)
VALUES(v1,v2,...)//显式的插入        

区别就是显式的假如需要写上表项的名字。

2.6 更新语句

UPDATE 关键字      
SET 关键字          

UPDATE TABLENAME
SET c1 = v1, c2 = v2, ...
WHERE condition;//这个不难看出来是什么意思

2.7 删除语句

DELETE 关键字   
TRUNCATE 关键字
DROP 关键字     
DELETE FROM TABLENAME               
WHERE condition;//删除tablename中满足condition的数据         

DROP TABLE;
//把table删除的一干二净，啥也不剩。

TRUNCATE TABLE;          
//删除内容，释放空间，但是表的定义保留。        

DELETE FROM test;               
DELETE * FROM test;               
//删除内容，但是不释放空间，要保留定义。

删除程度:DROP>TRUNCATE>DELETE
